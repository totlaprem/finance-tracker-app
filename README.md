# finance_tracker

A new Flutter project.
Finance Tracker App
An AI-powered personal finance tracker built using Flutter that helps users manage expenses, analyse spending patterns, and make smarter financial decisions.

Features
1. Add & track daily expenses
2. Income management
3. Category-wise expense tracking
4. Interactive charts & analytics
5. AI-based financial insights (optional if added)
6. Clean and modern UI

Tech Stack
1. Flutter (Dart)
2. SharedPreferences / Local Storage
3. Charts (fl_chart)
4. REST APIs
